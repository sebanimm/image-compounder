# 이미지 합성기

웹캠으로 얼굴과 사진을 합쳐보아요~~!

## 기능

- 사진 파일 선택 버튼 : 얼굴과 합성할 사진을 선택할 수 있습니다.

- 캡처 버튼 : 현재 합성된 이미지가 다운로드 폴더에 저장됩니다.
